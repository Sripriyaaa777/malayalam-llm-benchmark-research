\# Dataset Download Instructions



\## DravidianCodeMix Dataset



1\. Clone the dataset repository:

```bash

cd data

git clone https://github.com/bharathichezhiyan/DravidianCodeMix-Dataset.git

```



2\. The Malayalam sentiment files are in:

```

DravidianCodeMix-Dataset/DravidianCodeMix/

├── mal\_full\_sentiment\_train.csv

├── mal\_full\_sentiment\_dev.csv

└── mal\_full\_sentiment\_test.csv

```



\## Dataset Statistics



\- \*\*Total samples\*\*: 15,694

\- \*\*Clean samples\*\*: 9,323 (after filtering)

\- \*\*Positive\*\*: 6,363 (68%)

\- \*\*Negative\*\*: 2,047 (22%)

\- \*\*Mixed\_feelings\*\*: 913 (10%)



\## Note



Large data files are not included in this repository. 

Download them separately following the instructions above.
